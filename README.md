# Stanislav.jp — 東京でWeb制作・SEOを行う中で気づいたことを書き留めるノート

このリポジトリでは、日々の案件や相談の中で「これは他の人にも役に立ちそうだ」と思った内容を、ゆるくまとめています。  
テーマは主に **Web制作、SEO、コンテンツ、マーケティング、技術まわりのメモ** です。

普段お客様と話していると、同じような悩みや疑問が出てくることが多く、  
「だったら公開しておこう」と思ったのがこのリポジトリを作った理由です。

難しい専門用語はなるべく使わず、実務で迷いやすいポイントを中心に書いています。

## このノートで扱っている内容

- 東京の中小企業でよく起きるWebまわりの課題  
- LPやWordPressサイトを作る時の注意点  
- 検索順位が落ちた時の考え方  
- 東京のローカル検索で結果を出すための基礎  
- 実際の案件であった「よくあるミス」  
- 自分自身の失敗談や改善の過程  
- 技術的な設定メモ（自分の備忘録も兼ねています）

書いている内容は専門的ですが、できるだけ “ふつうの言葉” に落とし込むよう心がけています。

## 文章のスタイルについて

- AI っぽい書き方を避ける  
- 説明を詰め込みすぎない  
- 例や背景を少し添えて、読みやすく  
- 東京で実際に起きているケースを前提に  
- 広告っぽい空気をなくす  
- 長すぎず、短すぎず。自然な長さでまとめる  

SEOのことはよく扱いますが、「SEOのために書く文章」ではなく  
**“実際に人が読む前提の文章”** を意識しています。
